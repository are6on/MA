Task manager for a company.The manager gives to its employees task that they should finish until the deadline.The manager can see a statistic of all the tasks finished by an employee or all of them in a month. The manager has a list of all the employees.
A task has a name, a description and a deadline. The employees are notified when  receiving a task. The manager and the employees have pages where the information about them is showed, also here the manager and employees can see the task that respective employee has finished. Locally the employees can see the tasked he has to finish when he has a connection to the internet the data will be synchronized.
